Bienvenue sur la carte des bassins scolaires des écoles primaires de la région de Montréal

Cette carte vous aide à trouver le bassin scolaire d'une école primaire et ses indicateurs IMSE/SFR.

  1. Utilisez Rechercher une école pour trouver rapidement un établissement.
  2. Utilisez Ajouter un marqueur pour voir le bassin associé à cet endroit.
  3. Cliquez sur Ma position pour centrer la carte sur votre emplacement actuel.
  4. Utilisez la loupe pour rechercher votre adresse.
  5. Consultez le panneau de droite pour les détails et les graphiques.
  6. Utilisez Créer un itinéraire pour ouvrir le trajet dans Google Maps.
  7. Pour plus d'informations, consultez les liens disponibles dans le panneau d'information.
